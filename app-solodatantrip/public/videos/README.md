# Vídeo do hero

Coloque aqui `hero-rtk.mp4` (MP4, ~10–30 s, loop) com equipamentos usando RTK/NTRIP
(colheitadeira, drone, máquina agrícola, etc.).

Se o arquivo não existir, o site usa automaticamente um vídeo de colheita (Mixkit) pela internet.

Para usar outro URL: defina `NEXT_PUBLIC_HERO_VIDEO_URL` no `.env.local`.
